# CRIATHON-AOC
O projeto "arquitetUNO" adapta a mecânica do jogo UNO para o ensino de Arquitetura de Computadores (AOC). Ao substituir os números tradicionais por componentes de hardware, a ferramenta visa aumentar o engajamento dos alunos e facilitar a compreensão de conceitos complexos como MIPS e datapath.
